HTML significa hipertext markup language (lenguaje de marcado de hipertexto) y es el lenguaje con el cual se estructuran las páginas web. "Index" es el nombre que se le da a la página inicial de una web. Un "tag" o "etiqueta" es una caja en donde se deposita información. Las etiquetas lineales son aquellas que contienen todo en una linea, las adaptativas se ajustan al tamaño del contenedor.

En este caso utilizamos la etiqueta "a href" dentro de la etiqueta button.

Enlace a otra web

Los enlaces a otra web requieren de colocar el dominio completo (https://google.com).

Los enlaces pueden abrirse dentro de la misma pestaña, o bien abrirse en otra pestaña (target="_blank")

Metadatos

Un metadato es, como su nombre lo indica, un dato que sirve para que el navegador encuentre la página web mediante determinadas palabras o acciones.

Se escribe con la etiqueta <meta></meta>

Click aquí para aprender más acerca de Metadatos

Nav

La etiqueta <nav> sirve para colocar una barra de navegación en la web.

En este caso existe un nav en la parte superior de la página (inicio, campus, registro)

Article y Section

Estos apartados se encuentran escritos dentro de secciones, que en su conjunto pertenecen a un articulo.

Sirven para ordenar la información dentro de las páginas web, nos servirá para ajustar la información cuando vayamos al diseño.

Nuevamente, se puede pensar HTML como cajas. Estas cajas se denominan etiquetas. En este caso serían las etiquetas <article> y <section>

Aside

La etiqueta <aside> es otra caja. Por lo general se ubica en la parte derecha de la página.

Footer

El <footer> es el pie de página, por lo general se ubican los enlaces de redes sociales y contactos.

CSS

Las hojas de estilo en cascada (CSS, cascading style sheets) permiten crear páginas web atractivas.

CSS es una forma de aplicar diseño a nuestras páginas web.

Para aplicar un archivo CSS en un archivo HTML y que efectivamente se vean los cambios en el diseño del documento, debemos colocar:

link rel="stylesheet" type="text/css" href="archivo.css"
